# Ephoria
Ephoria | Discord Bot [1.1.0]

Эйфория - Мультисерверный бот для Discord с поддержкой MySQL/MonoDB баз данных.

Написан на DiscordJS версии 12+.
Написан абсолютно с нуля, но в некоторых моментах используются заготовки из fatCat бота адаптированные под версию DiscordJS 12. 

Текущие возможности:
* Смена префикса локально для сервера
* Бан человека с заносом информации в MySQL БД
* Регистрация человека в БД когда он заходит на сервер 
* Удаление информации о человеке когда он покидает сервер
* Красивое окно информации о боте 
* Различные веселые команды (8ball, cats, doge)
* Проверка пинга и отклика API бота
* Обновление статуса сервера в профиле бота


Бот реализован на структурной системе StateManager что позволяет удалять и добавлять различные категории команд и сами команды не затрагивая другие команды и не мешая адекватной работе бота.
